Welcome to Smart Contact Manager

A Cloud contact management system

Technologies used: Java, Spring Boot, JavaScript, MySQL Deployed using Amazon web services.

Features:

  • Supports CRUD operations for contacts.
  • Image upload using cloudinary.
  • Secure login using Spring security.
  • Supports signin with google and github using OAuth2.
  • Email verification for additional authentification.
